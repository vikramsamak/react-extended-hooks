# `usePlatformDetails` Hook

The `usePlatformDetails` hook provides information about the user's platform. It returns the following details:

- **userAgent**: The user agent string of the browser/platform.
- **language**: The browser's language preference.
- **onLine**: The user's current online status (true for online, false for offline).

## Example Usage

Hereâ€™s an example of how to use the `usePlatformDetails` hook in your React component:

```tsx
import React from "react";
import { usePlatformDetails } from "react-util-hooks";

const MyComponent: React.FC = () => {
  const { userAgent, language, onLine } = usePlatformDetails();

  return (
    <div>
      <h3>Platform Details:</h3>
      <ul>
        <li>
          <strong>User Agent:</strong> {userAgent}
        </li>
        <li>
          <strong>Language:</strong> {language}
        </li>
        <li>
          <strong>Online Status:</strong> {onLine ? "Online" : "Offline"}
        </li>
      </ul>
    </div>
  );
};

export default MyComponent;
```
